## 📟 Setup Environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
pip install pipreqs
pip install streamlit
```

## 🚀 Cara Menjalankan
Jalankan Streamlit dengan perintah berikut:
```
streamlit run submission1.py
```
